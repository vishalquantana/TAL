# Touch A Life (TAL) Portal

## Project Overview

The TAL Portal is a web-based platform designed to manage volunteers, students, donors, and administrators under the Touch A Life Foundation. 

It aims to digitize student data collection, improve transparency, and streamline scholarship management processes.

---

## Tech Stack

- React (Vite)
- Supabase (Database & Authentication)
- Node.js (if applicable)

---

## Setup Instructions

1. Clone the repository:
   git clone https://github.com/himasree1111/TAL.git

2. Install dependencies:
   npm install

3. Create a `.env` file and add:
   VITE_SUPABASE_URL=your_url
   VITE_SUPABASE_ANON_KEY=your_key

4. Run the project:
   npm run dev

---

## Note

The `.env` file is not included in the repository for security reasons.
